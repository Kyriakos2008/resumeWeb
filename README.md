# Kyriakos Nikoletti – Portfolio Website

Personal portfolio and resume website built with Astro and Tailwind CSS.

## About
This website showcases my projects, achievements, technical skills, and interests in software development, engineering, and creative media.

Built as a fast and responsive personal hub for:
- Project showcases
- Technical experience
- Education & achievements
- Tools and technologies
- Future blog posts and utilities

## Tech Stack
- Astro
- Tailwind CSS
- TypeScript
- Starwind UI

## Featured Projects

- Receipt Generator

## Features
- Responsive design
- Dark theme support
- Clean minimal UI
- Organized skills and achievements sections
